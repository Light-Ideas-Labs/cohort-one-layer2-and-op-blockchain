# Repository Guide

## Overview

This guide provides instructions on how to use and contribute to the Continuum Block repositories.

## Structure

The repository is organized into the following directories:

- **/docs**: Documentation for the course, including this guide, FAQs, and additional resources.
- **/lessons**: Lesson materials, including slides, code examples, and assignments.
- **/projects**: Project templates and examples for hands-on activities.
- **/resources**: Access additional resources such as recommended readings, reference links, and useful tools.
